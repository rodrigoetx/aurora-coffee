# ☕ Aurora Coffee: High-Conversion Boutique UI

> **Blending premium aesthetics with accessible UI through strategic Agentic Development.**

## 📖 Context
A boutique coffee e-commerce UI focused on maximizing high conversion rates while maintaining an elegant, warm-toned brand identity. Designed to serve as a frictionless gateway for artisanal coffee subscriptions.

## 🤖 Vibe Coding Workflow: Next.js & Tailwind CSS
This interface was built using a highly focused **Vibe Coding** pipeline:
*   **Structured Prompting:** Used precise, intent-driven prompts to prototype the 'Roast Switcher' (a dynamic dark/light mode functionality).
*   **Manual Validation (HITL):** Audited the AI-generated Tailwind classes and component architecture to guarantee WCAG accessibility standards and flawless visual appeal across all viewports.
*   **Rapid Scaffolding:** Directed the AI to build the foundational layout, allowing me to focus exclusively on component encapsulation and state management.

## ✨ Key Features
| Feature | Technical Implementation | Impact |
| :--- | :--- | :--- |
| **Roast Switcher** | Next.js State + Tailwind Dark Mode | Instant, theme-aware transitions tailored to user preference. |
| **Premium Scroll** | Framer Motion Viewport Hooks | Engaging, staggered entry animations for product discovery. |
| **Slide-Out Cart** | Glassmorphism UI | A visually pleasing, non-intrusive purchasing experience. |
